# TFT & Ultrasonic

Test project for displaying data from Ultrasonic range sensor on a TFT display

## Tested with
- Arduino DUE R3
- 2.8" Adafruit TFT LCD Shield (v1.1)
- RadioShack Ultrasonic Range Sensor (signal wire connected to pin 53)
